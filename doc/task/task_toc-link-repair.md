## 1. 任務目標 (Objective)

在執行「全站 FAQ 內容擴展與結構升級」後，發現部分更新後的頁面，其側邊欄的目錄 (TOC) 中「常見問題 (FAQ)」的主連結並未正確導向頁面內對應的 `<h2>常見問題</h2>` 標題。本任務旨在快速修復這些失效的主連結，確保使用者點擊目錄項目時能順利跳轉至正確的內容位置，同時確保不對頁面的其他內容或結構產生非預期的改動。

**重要說明**: 本任務僅專注於修正 TOC 中指向「常見問題」區域的主連結。對於 FAQ 內部各個問題的細部 ID 連結，將不進行檢查或修改。

## 2. 實作策略 (Implementation Strategy)

### 2.1. 技術方案

- **技術**: 純 HTML 修改。
- **原則**: 精準定位，最小修改。僅修正 `<a>` 標籤的 `href` 屬性，使其指向對應 `<h2>` 標題的 `id`，或在必要時補全遺失的TOC結構。

### 2.2. 檔案結構

此任務已修改先前根據 `task_global-faq-strategy.md` 更新過的文章頁面，主要集中在 `post/` 目錄下。
